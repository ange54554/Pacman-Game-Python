# pacman-python-game
Pacman game (Python)

# Description
This project is a Pacman-style game developed in Python as part of a structured academic project for a software engineering course.
While the overall objectives were provided, all core features, game logic, and behaviours (movement, chase, collisions) were implemented independently.


The game includes player movement, enemies (ghosts), collectibles and maze navigation.

# Features
-Player-controlled movement and collision detection
-Rule-based enemy behaviour:
  -Red ghost chases player
  -Pink ghost attempts to intercept the player
  -Blue ghost follows a random pattern
  -Orange ghost alternates between behaviors
-Score and collectibles system (dots and "energizers")
-Maze-based gameplay

# Technologies
- Python
- Object-Oriented Programming
- Pygame
